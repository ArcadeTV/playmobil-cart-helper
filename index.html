Playmobil Warenkorb-Helfer

Automatisiere das Hinzufügen mehrerer Artikel zum Warenkorb

⚠️ Wichtige Anleitung

  1. Öffne playmobil.com in einem neuen Tab und melde dich an (falls nötig)
  2. Gib unten deine Artikelliste ein
  3. Klicke auf "Script generieren"
  4. Kopiere das generierte Script
  5. Öffne die Browser-Konsole auf playmobil.com (F12 → Console)
  6. Füge das Script ein und drücke Enter
Format: Eine Zeile pro Artikel: Artikelnummer, Menge
Beispiel: 30238993, 2 fügt 2x Artikel 30238993 hinzu
Empfohlen: 1500-3000ms
